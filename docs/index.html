Video → Data

An end-to-end pipeline that converts human demonstration videos into simulation-ready assets and physics-grounded robot training data.

End-to-end workflow

  1. 1

    Video Ingestion Agent

    LangGraph workflow that segments demos into action clips, extracts an entity-relation scene graph, and stores SigLIP-2 frame embeddings.

    video_ingestion_agent/
  2. →
  3. 2

    Reconstruction

    Containerized vision modules turn selected clips into per-frame depth, masks, textured meshes, 6-DoF object poses, and SMPL human body.

    reconstruction/
  4. →
  5. 3

    Robotic Grounding

    Retarget human motion onto the target embodiment, then drive Isaac Lab environments trained with RSL-RL PPO to produce deployable policies.

    robotic_grounding/
